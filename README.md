# opendata-campus-mcp

教育資源導航 MCP — 以 TWCampus 作為目錄入口，路由至台灣官方教育平台搜尋。

## MCP 工具

| 工具 | 說明 |
|------|------|
| `discover_education_sources` | 從 TWCampus 目錄找到相關官方教育平台 |
| `browse_education_source` | 在指定官方教育平台上搜尋學習資源 |
| `read_education_resource` | 讀取單一公開教育資源頁面摘要 |

## 安裝

```bash
pip install -e ".[dev]"
```

## 執行

```bash
opendata-campus-mcp
```
